Daily

Current Tokyo Night defaults. Keep full AGS widgets, normal Waybar, normal animations, restored wallpaper state.

ThinkPad Red

TrackPoint-inspired profile for screenshots: red accents, darker panels, active workspace underline, lockscreen red trim.

Battery

Respect TLP policy: static wallpaper, reduced Hyprland animation, no video wallpaper, AGS shows battery-first controls.

Showcase

One command opens a curated layout with fastfetch, btop, Yazi, Neovim, AGS dashboard, and optional screen capture.

Implementation convention: Treat desktop modes as state plus reload hooks. Do not add a new theme framework. Stylix remains the declarative source; runtime modes adjust generated AGS/Waybar CSS, Hyprland keywords, and wallpaper state.
