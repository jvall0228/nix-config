Avoid redundant UX frameworks

Give every surface one job.

The cleanest path is not adding tools. It is making the current tools stop competing: Walker searches, Rofi performs scoped tasks, AGS owns interactive state, Waybar reports status, Hyprlock owns the lock surface, and Stylix owns durable theming.
