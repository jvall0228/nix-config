Add to current dashboard

Desktop mode tile, capture state, TLP/charge thresholds, NVIDIA runtime state, Syncthing status, Tailscale state, and latest screenshot/recording shortcut.

Remove from current dashboard

Separate direct screen recording to /tmp. It competes with the capture menu. AGS should stop, reveal, or show state for the canonical recorder.

Keep as separate popups

Calendar, audio mixer, network, Bluetooth, media, and notifications already map cleanly from Waybar clicks to AGS popups.

Use mode-aware density

Daily mode can stay rich. Battery/focus modes should collapse less important sections and reduce polling-heavy visual surfaces.
